Email Manager (OOP in Python)

A Python application demonstrating object-oriented programming with GUI and console interfaces

Overview

A Python-based email management system built using object-oriented programming principles. It features both a console interface and a graphical user interface, enabling users to create, edit, delete, and organize emails efficiently. The project showcases core OOP concepts such as inheritance, encapsulation, and polymorphism, and emphasizes modular design.

Technologies Used

Key Features

  • Object-oriented design with inheritance and encapsulation
  • Both console and GUI interfaces
  • Create, edit, delete, and organize emails
  • Data persistence with file storage

Challenges & Solutions

Designing flexible class hierarchy: Used inheritance to reduce code duplication and promote reusability.
Implementing GUI and console together: Separate core logic from interface code; used design patterns to ensure modularity.
